## Component Structure

### Reusable Components
1. Button (primary, secondary, outline, ghost, danger)
2. Card (default, hover, interactive)
3. Input (text, email, password, textarea)
4. Select / Dropdown
5. Modal / Dialog
6. Sidebar
7. Navbar
8. Badge / Tag
9. Avatar
10. Progress Bar
11. Chart Container
12. Table
13. Alert / Toast
14. Checkbox / Radio
15. Toggle / Switch

## Layout Grid

- Mobile: 4px padding, single column
- Tablet: 24px padding, 8-column grid
- Desktop: 32px padding, 12-column grid
- Max width: 1440px

## Animation Principles

```
Easing: ease-in-out, cubic-bezier(0.4, 0, 0.2, 1)
Hover: scale-105, brightness-110
Active: scale-95
```
