# SKILLSYNC-AI-INTELLIGENT-CAREER-PATHWAY-GENERATOR-USING-RESUME

SkillSync AI – An Intelligent Career Pathway Generator Using Resume Analytics is a full-stack MERN platform that analyzes resumes, detects skill gaps, and produces personalized learning paths using AI-driven recommendations.
The system extracts user skills through OpenAI-powered NLP models and compares them with industry job requirements to generate learning roadmaps.
A secure Node.js/Express backend handles authentication, user data, AI processing, and integrates with MongoDB Atlas for scalable storage.
The frontend is built with React and Tailwind CSS, delivering a modern SaaS-like UI with smooth interactions and responsive layouts.
Users upload their resumes, after which the platform provides job match percentages, skill insights, and personalized weekly study plans.
The workflow includes 
.
resume parsing → skill extraction → gap analysis → dynamic roadmap generation → job recommendation.
.
Advanced modules include an AI interview coach, interactive dashboard analytics, and real-time progress tracking.
JWT-based authentication ensures secure access, while REST APIs allow smooth communication between frontend and backend.
The entire system follows a structured MERN architecture, with modular controllers, services, and reusable UI components.
Deployment uses Vercel for the frontend and Render/AWS for the backend, making the platform globally accessible and cloud-ready.
